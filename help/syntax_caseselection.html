Filters across different fields are AND'd together. e.g. tag:gaia suite:smoketest => tag:gaia AND suite:smoketest

Tags filters are OR'd together. e.g. tag:gaia tag:rtl suite:smoketest=> (tag:gaia OR tag:rtl) AND suite:smoketest. Issue 77 is trying to add AND support for tags
